# Infinity-ammo-OX-Inventory-
This FiveM script gives players infinite ammo for all weapons in their ox_inventory.

## Features
- **Infinite Ammo:** Automatically fills ammo for all weapons.
- **Lightweight & Efficient:** Minimal performance impact.
- **Plug & Play:** Works out-of-the-box with ox_inventory.
- **Fully Compatible:** Designed for FiveM servers using ox_inventory.
